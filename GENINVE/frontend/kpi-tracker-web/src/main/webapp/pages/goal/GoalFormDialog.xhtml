Description
 Organisation goals are top-level goals that don't require a parent. They set the overall direction for the organization. Department goals must have a parent organisation goal and specify which department they belong to. Team goals must have a parent department goal. The department will be automatically set from the parent goal. Individual goals must have a parent team goal and specify the individual owner. The department will be automatically set from the parent goal.
